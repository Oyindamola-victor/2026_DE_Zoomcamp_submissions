# 2026 Data Engineering Zoomcamp – Submissions

Homework submissions by week.

- **Week_1** – Ingest & Terraform
- **Week_2** – Workflow orchestration (e.g. Kestra)
- **Week_3** – Data warehouse (BigQuery, GCS load script)

## Pushing updates

**Do not push directly to `main`.** Use a branch and merge (e.g. via PR):

```bash
git checkout -b week-N-description   # or your branch name
git add .
git commit -m "Your message"
git push -u origin week-N-description
```

Then open a Pull Request on GitHub and merge into `main`.
